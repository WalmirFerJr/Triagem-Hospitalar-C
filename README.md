# Triagem-Hospitalar-C

Projeto desenvolvido como trabalho da faculdade no segundo semestre na disciplina de Introdução e Análise de Algoritmos

• Desenvolvi um sistema de triagem hospitalar em Linguagem C para automatizar a classificação de prioridade de pacientes com base em cores e tempo de espera. • Implementei algoritmos de ordenação customizada (qsort) e lógica de escalonamento dinâmico, que reclassifica a gravidade do paciente conforme o tempo de espera aumenta. • Apliquei conceitos avançados de alocação dinâmica de memória e manipulação de arquivos (CSV) para processar grandes volumes de dados de atendimento de forma eficiente.
